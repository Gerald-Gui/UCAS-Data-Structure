# UCAS-Data-Structure

#### 介绍
source codes for UCAS CS data structure course, including ADTs, OJ assignments and projects

#### ADT

ADT的编写主要参考课程所用教材^[严蔚敏，吴伟民：数据结构（C语言版），清华大学出版社]中的ADT要求，但不完全根据教材进行编写，视需求选择性编写函数/方法。ADT的push与本地编写情况同步。

#### OJ assignments

**此部分仅供参考，请勿用于不良行为。**
